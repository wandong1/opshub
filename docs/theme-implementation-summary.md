## 技术实现细节

### 主题切换机制
```typescript
// 通过 body 属性控制 Arco Design 深色主题
document.body.setAttribute('arco-theme', 'dark')  // 深色
document.body.removeAttribute('arco-theme')       // 浅色
```

### 跟随系统主题
```typescript
const mediaQuery = window.matchMedia('(prefers-color-scheme: dark)')
mediaQuery.addEventListener('change', (e) => {
  if (theme === 'auto') {
    applyTheme(e.matches ? 'dark' : 'light')
  }
})
```

### localStorage 存储
```typescript
{
  "opshub_theme": "auto",        // light | dark | auto
  "opshub_layout": "sidebar",    // sidebar | topbar
  "opshub_sidebar_collapsed": false
}
```

## 使用指南

### 切换主题
1. 点击顶栏右侧的主题图标按钮（太阳/月亮/电脑）
2. 在弹出的抽屉中选择主题模式：
   - **浅色**：固定使用浅色主题
   - **深色**：固定使用深色主题
   - **跟随系统**：自动根据操作系统主题切换

### 切换布局
1. 点击顶栏右侧的主题图标按钮
2. 在"布局模式"区域选择：
   - **侧边栏**：传统的左侧侧边栏 + 顶部导航
   - **顶部导航**：现代的顶部水平菜单布局
3. 切换后页面会自动刷新应用新布局

### 重置设置
1. 在主题设置抽屉底部点击"重置为默认设置"
2. 将恢复为：浅色主题 + 侧边栏布局 + 侧边栏展开

## 性能优化

1. **CSS 变量**：使用 CSS 变量实现主题切换，无需重新加载样式
2. **过渡动画**：0.3s 平滑过渡，提升用户体验
3. **localStorage**：本地存储用户偏好，减少服务器请求
4. **条件渲染**：根据布局模式按需渲染组件

## 已知限制

1. **插件页面**：部分插件页面可能需要单独适配深色主题
2. **第三方组件**：xterm.js、ECharts 等可能需要额外配置
3. **图片资源**：深色模式下某些图片可能需要反色处理
4. **布局切换**：需要刷新页面才能应用新布局（避免状态丢失）
