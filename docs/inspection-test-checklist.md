### 3. 测试执行功能

### 4. 执行策略配置
- [ ] 创建巡检组时可以选择执行策略（并发/顺序）
- [ ] 选择并发执行时，显示并发数量输入框

- [ ] 并发数量默认值为50
- [ ] 并发数量限制在1-200之间

### 5. 列表展示
