# Web站点管理页面UI优化

## 修复问题

### 1. Agent主机列表查询问题
**问题描述**: 新增站点时，Agent主机列表查询不到数据

**原因**: 数据获取逻辑不一致，有些接口返回 `data.list`，有些直接返回 `list`

**解决方案**: 统一使用兼容性写法
```typescript
const list = res.data?.list || res.list || []
```

### 2. 复制凭据功能优化
**原有方式**: 单个按钮复制所有凭据信息（URL、账号、密码）

**优化后**: 三个独立按钮，通过下拉菜单展示
- 复制URL - 直接复制站点访问地址
- 复制账号 - 复制访问用户名
- 复制密码 - 从后端获取完整信息后复制密码

**优势**:
- 更灵活，用户可以按需复制单项信息
- 避免一次性复制大量文本
- 更符合实际使用场景

## UI重构

### 从表格改为卡片展示

**原有方式**: 使用 `a-table` 表格展示站点列表

**优化后**: 使用响应式卡片网格布局

#### 卡片结构
```
┌─────────────────────────────────┐
│ 卡片头部（渐变背景）              │
│ ┌────┐                          │
│ │图标│ 站点名称                  │
│ └────┘ [类型] [状态] [Agent]    │
├─────────────────────────────────┤
│ 卡片内容                         │
│ 访问地址: https://...           │
│ 备注: ...                       │
│ 业务分组: [标签1] [标签2]        │
│ Agent主机: [主机1] [主机2]       │
├─────────────────────────────────┤
│ 卡片底部（操作按钮）              │
│ [访问站点] [复制凭据▼] [编辑] [删除] │
└─────────────────────────────────┘
```

#### 视觉特性
- **响应式网格**: `grid-template-columns: repeat(auto-fill, minmax(380px, 1fr))`
- **悬停效果**: 鼠标悬停时卡片上浮并显示阴影
- **渐变背景**: 卡片头部使用浅色渐变，图标使用紫色渐变
- **大图标**: 56x56px 圆角图标，支持 emoji 和图片
- **标签展示**: 使用 Arco Tag 组件展示类型、状态、分组等信息
- **信息分层**: 清晰的头部、内容、底部三层结构

#### 交互优化
- **空状态**: 无数据时显示友好的空状态提示
- **加载状态**: 加载时显示居中的 Spin 组件
- **下拉菜单**: 复制凭据使用 Dropdown 组件，提供三个选项
- **分页**: 底部居中显示分页器

## 技术细节

### 新增图标
```typescript
import {
  IconUser,    // 用户图标（复制账号）
  IconLock     // 锁图标（复制密码）
} from '@arco-design/web-vue/es/icon'
```

### 样式变量
使用项目统一的 CSS 变量：
- `--ops-primary`: 主色调
- `--ops-text-primary/secondary`: 文字颜色
- `--ops-border-color`: 边框颜色
- `--ops-content-bg`: 内容区背景色

### 响应式设计
- 最小卡片宽度: 380px
- 自动填充可用空间
- 卡片间距: 20px
- 支持多列布局（根据屏幕宽度自动调整）
