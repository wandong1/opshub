# 巡检管理测试执行日志优化说明

## 优化概述

优化了巡检管理中测试按钮的执行日志显示，增加了更详细的信息，方便用户快速定位和分析测试结果。

## 优化内容

### 1. 新增巡检项名称显示

**优化前**：日志中只显示主机名和 IP，无法区分是哪个巡检项
**优化后**：在日志头部显著位置显示巡检项名称

### 2. 新增执行类型标签

增加了执行类型的可视化标签，方便快速识别：
- **命令执行** (蓝色)
- **脚本执行** (紫色)
- **PromQL** (青色)
- **网络拨测** (橙色)

### 3. 优化拨测类型显示

**优化前**：拨测类型没有主机信息，显示为空
**优化后**：拨测类型显示"网络拨测"标识

### 4. 新增可展开的详细信息

点击"查看详情"可展开查看：

#### 命令执行类型
- 执行命令
- 超时时间
- 断言类型
- 断言阈值

#### 脚本执行类型
- 脚本类型 (shell/python/bash 等)
- 脚本参数
- 超时时间
- 断言类型
- 断言阈值

#### PromQL 类型
- PromQL 查询语句
- 超时时间
- 断言类型
- 断言阈值

#### 网络拨测类型
- 拨测配置 (类别 - 类型)
- 超时时间
- 断言类型
- 断言阈值

## 技术实现

   - 增加可展开的详细信息区域
   - 优化拨测类型的显示逻辑

2. **脚本改动**：
   - 修改 `handleTestRun` 函数，增加详细信息映射
   - 修改 `handleTestRunItem` 函数，增加详细信息映射
   - 新增 `getExecutionTypeText` 函数
   - 新增 `getExecutionTypeColor` 函数

   - 新增 `.log-details-toggle` 样式
   - 新增 `.log-details` 样式
   - 新增 `.detail-item` 样式

## 使用说明

1. 在巡检管理页面，点击巡检组的"测试"按钮
2. 或在编辑巡检组时，点击单个巡检项的"测试"按钮
3. 在弹出的测试执行日志窗口中查看详细信息
4. 点击"查看详情"可展开查看执行配置和断言信息
5. 点击"收起详情"可折叠详细信息

## 注意事项

1. 详细信息仅在有配置时显示（如命令、脚本参数、PromQL 查询等）
2. 拨测类型不显示主机信息，统一显示"网络拨测"
3. 详细信息默认折叠，需要手动展开查看
4. 执行类型标签颜色固定，便于快速识别

## 兼容性

- 前端：Vue 3 + Arco Design Vue
- 后端：Go 1.25+
- 浏览器：Chrome 90+, Firefox 88+, Safari 14+
