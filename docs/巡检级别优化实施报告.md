## 🎨 UI 设计规范

### 级别标签颜色方案

**巡检级别：**
- 高：红色 `red` (#f53f3f)
- 中：橙色 `orange` (#ff7d00)
- 低：绿色 `green` (#00b42a)

**风险等级：**
- 高：红色 `red` (#f53f3f)
- 中：橙红色 `orangered` (#ff5722)
- 低：蓝色 `blue` (#165dff)

### 标签使用规范

```vue
<a-tag :color="getLevelColor(record.inspectionLevel)" size="small">
  {{ getLevelText(record.inspectionLevel) }}
</a-tag>
```
